## Run Locally

Clone the project

```bash
  git clone https://github.com/Abhinav2011/my-portfolio
```

Go to the project directory

```bash
  cd my-portfolio
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
